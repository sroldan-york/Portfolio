# Santiago's Portfolio

A personal portfolio website built with HTML, CSS, and Bootstrap 5.

## Sections

- **About Me** — Introduction and soft skills (Problem Solving, Teamwork, Communication, Leadership, Adaptability, Critical Thinking)
- **Projects** — Showcase of 4 projects with tech stack badges and links to GitHub and live demos
- **Skills** — Interactive carousel displaying technical skills: Python, Java, and Linux
- **Contact** — Contact form and links to Gmail, GitHub, and LinkedIn

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript
